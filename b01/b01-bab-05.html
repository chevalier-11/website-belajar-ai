5.1 Percabangan: if, elif, else

Percabangan adalah kemampuan program untuk memilih jalan yang berbeda berdasarkan sebuah kondisi. Kalau kamu bayangkan kode sebagai petunjuk perjalanan, percabangan adalah bagian "jika lampu merah, berhenti; jika lampu hijau, jalan." Di Python, ini ditulis dengan kata kunci if, elif (singkatan dari "else if"), dan else.

Satu hal yang sangat penting di Python adalah indentasi — baris kode di dalam blok if harus menjorok ke dalam sebesar 4 spasi. Python menggunakan indentasi untuk menentukan blok kode, bukan kurung kurawal seperti bahasa lain. Ini berarti indentasi bukan soal estetika semata — ia adalah bagian dari sintaksis Python yang wajib benar.

python 
# Struktur dasar if-elif-else
nilai = 78

if nilai >= 90:
    print("Grade: A")
elif nilai >= 80:
    print("Grade: B")
elif nilai >= 70:
    print("Grade: C")    # ← ini yang tercetak
elif nilai >= 60:
    print("Grade: D")
else:
    print("Grade: E — tidak lulus")

# Python memeriksa kondisi dari atas ke bawah
# dan berhenti begitu menemukan satu yang True

Python memeriksa setiap kondisi secara berurutan dari atas. Begitu menemukan kondisi yang bernilai True, ia menjalankan blok kode yang sesuai lalu melompati semua elif dan else di bawahnya. Kalau tidak ada satu pun kondisi yang True, blok else-lah yang dijalankan. Blok else ini opsional — kamu tidak wajib menyertakannya.

⚠️
Indentasi Salah = Program Error

Di Python, indentasi yang salah akan menyebabkan IndentationError yang menghentikan program. Gunakan 4 spasi secara konsisten — bukan campuran Tab dan spasi. VS Code otomatis mengubah Tab menjadi 4 spasi untuk file Python, jadi kamu tidak perlu khawatir selama pakai editor yang benar.

Kondisi Majemuk dan Kondisi Bersarang

Kondisi bisa diperkaya dengan menggabungkan operator logika and, or, dan not, atau dengan menaruh blok if di dalam blok if yang lain (bersarang). Dalam kode kecerdasan buatan, kamu sering menemui kondisi seperti "jika akurasi lebih dari 85% DAN jumlah data lebih dari 1000, maka model siap digunakan":

python 
# Kondisi majemuk — lebih bersih dari bersarang
akurasi     = 0.87
jumlah_data = 1500

if akurasi >= 0.85 and jumlah_data >= 1000:
    print("Model siap deploy: akurasi tinggi + data cukup")
elif akurasi >= 0.85 and jumlah_data < 1000:
    print("Akurasi bagus, tapi data masih kurang")
else:
    print("Model perlu ditingkatkan")

# Perbandingan berantai — idiom khas Python yang elegan
nilai = 75
if 60 <= nilai < 80:
    print("Nilai berada di rentang C")  # ← tercetak

Ekspresi Kondisional Satu Baris

Python punya cara singkat menulis if-else sederhana dalam satu baris, sering disebut operator ternary. Sintaksinya: nilai_jika_true if kondisi else nilai_jika_false. Ini sangat berguna untuk penugasan nilai yang bergantung pada kondisi sederhana, dan sangat sering muncul dalam kode preprocessing data:

python 
# Penugasan kondisional satu baris
nilai  = 75
status = "lulus" if nilai >= 60 else "tidak lulus"
print(f"Status: {status}")  # → Status: lulus

# Membuat label dari probabilitas prediksi model
prob  = 0.73
label = "positif" if prob >= 0.5 else "negatif"
print(f"Prediksi: {label}")   # → Prediksi: positif

# Bisa langsung di dalam f-string
stok = 0
print(f"Stok: {'tersedia' if stok > 0 else 'habis'}")
# → Stok: habis

5.2 Perulangan: for Loop

Bayangkan kamu punya dataset dengan 10.000 baris dan perlu memproses setiap barisnya. Tanpa perulangan, kamu harus menulis 10.000 baris kode yang hampir identik — jelas tidak masuk akal. for loop memungkinkan kamu menjalankan blok kode yang sama untuk setiap elemen dalam sebuah koleksi: list, tuple, string, dictionary, atau objek yang bisa diiterasi lainnya.

python 
# Iterasi atas list
kota = ["Jakarta", "Bandung", "Surabaya", "Medan"]
for nama_kota in kota:
    print(f"Kota: {nama_kota}")

# Iterasi atas string — setiap karakter
for huruf in "Python":
    print(huruf, end=" ")
# → P y t h o n

# Iterasi atas dictionary dengan .items()
config = {"lr": 0.001, "batch": 32, "epoch": 50}
for kunci, nilai in config.items():
    print(f"  {kunci} = {nilai}")

Fungsi range() untuk Perulangan Angka

Sangat sering kamu butuh melakukan sesuatu sebanyak N kali atau mengiterasi rentang angka tertentu. Fungsi range() menghasilkan urutan angka yang bisa langsung dipakai oleh for. Ada tiga bentuk penggunaannya: range(stop), range(start, stop), dan range(start, stop, step). Penting diingat bahwa nilai stop tidak ikut masuk ke urutan — jadi range(5) menghasilkan 0, 1, 2, 3, 4 (bukan 5).

python 
# range(stop) — 0 sampai stop-1
for i in range(5):
    print(i, end=" ")
# → 0 1 2 3 4

# range(start, stop)
for i in range(1, 6):
    print(i, end=" ")
# → 1 2 3 4 5

# range(start, stop, step) — dengan langkah
for i in range(0, 20, 5):
    print(i, end=" ")
# → 0 5 10 15

# Mundur dengan step negatif
for i in range(10, 0, -2):
    print(i, end=" ")
# → 10 8 6 4 2

# Simulasi epoch pelatihan model
print("\n--- Simulasi Pelatihan ---")
for epoch in range(1, 6):
    loss = 1.0 / epoch
    print(f"Epoch {epoch}/5 — Loss: {loss:.4f}")
# → Epoch 1/5 — Loss: 1.0000
# → Epoch 2/5 — Loss: 0.5000
# → ... dan seterusnya

enumerate() dan zip() — Wajib Kamu Kuasai

Dua fungsi bawaan ini sangat sering muncul dalam kode Python profesional. enumerate() memberikan nomor indeks sekaligus nilainya saat iterasi — sangat berguna ketika kamu butuh tahu posisi elemen. zip() memungkinkan kamu mengiterasi dua atau lebih list secara bersamaan, elemen per elemen — sangat berguna saat kamu punya dua list yang berkorespondensi, seperti list nama model dan list akurasinya:

python 
# enumerate() — indeks + nilai sekaligus
model_list = ["Regresi Logistik", "Random Forest", "XGBoost"]

for nomor, nama in enumerate(model_list, start=1):
    print(f"{nomor}. {nama}")
# → 1. Regresi Logistik
# → 2. Random Forest
# → 3. XGBoost

# zip() — iterasi dua list bersamaan
nama_model = ["Reg. Logistik", "Random Forest", "XGBoost"]
akurasi    = [0.82, 0.91, 0.93]

for nama, skor in zip(nama_model, akurasi):
    print(f"{nama:20} → {skor:.1%}")
# → Reg. Logistik        → 82.0%
# → Random Forest        → 91.0%
# → XGBoost              → 93.0%

# Kombinasi enumerate + zip
fitur = ["usia", "gaji", "pendidikan"]
bobot = [0.3, 0.5, 0.2]
for i, (f, b) in enumerate(zip(fitur, bobot)):
    print(f"Fitur [{i}] {f}: bobot = {b}")
💡
Hindari range(len(list)) — Pakai enumerate()

Pemula sering menulis for i in range(len(daftar)): lalu mengakses daftar[i]. Ini bekerja, tapi bukan cara idiomatik Python. Cara yang lebih bersih dan profesional adalah for i, item in enumerate(daftar):. Kode yang mengikuti gaya Python disebut "pythonic" dan lebih mudah dibaca oleh kolega di tim.

5.3 Perulangan: while Loop

Sementara for mengiterasi atas koleksi yang sudah diketahui isinya, while terus berjalan selama sebuah kondisi bernilai True — tidak peduli sudah berapa iterasi. Kamu pakai while ketika kamu tidak tahu sebelumnya berapa kali perulangan perlu dilakukan.

Skenario paling umum untuk while di dunia kecerdasan buatan adalah pelatihan model konvergen: "terus latih sampai loss tidak turun signifikan lagi." Skenario lain termasuk membaca antrian data sampai kosong, atau meminta input ulang sampai nilai yang valid diterima.

python 
# while loop dasar
hitung = 1
while hitung <= 5:
    print(f"Hitungan: {hitung}")
    hitung += 1   # WAJIB! tanpa ini loop tidak pernah berhenti

# Simulasi pelatihan sampai konvergen
import random
random.seed(42)

loss        = 1.0
epoch       = 0
target_loss = 0.15

print("Mulai pelatihan...")
while loss > target_loss:
    penurunan = random.uniform(0.1, 0.3)
    loss      = max(loss - penurunan, 0)
    epoch    += 1
    print(f"  Epoch {epoch:2d}: loss = {loss:.4f}")

print(f"Konvergen setelah {epoch} epoch!")

Validasi Input dengan while Loop

Pola klasik yang sangat berguna: program terus meminta input sampai pengguna memasukkan nilai yang valid. Ini menggabungkan while dengan penanganan error menggunakan try-except — yang akan kita bahas lengkap di Bab 7:

python 
# Terus minta input sampai angka valid (0–100) diberikan
nilai = None
while nilai is None:
    teks = input("Masukkan nilai ujian (0–100): ")
    try:
        angka = int(teks)
        if 0 <= angka <= 100:
            nilai = angka
        else:
            print("Nilai harus antara 0 dan 100. Coba lagi.")
    except ValueError:
        print("Itu bukan angka. Coba lagi.")

print(f"Nilai diterima: {nilai}")
⚠️
Awas: Infinite Loop

Loop tak terbatas terjadi ketika kondisi while tidak pernah menjadi False — biasanya karena lupa mengubah variabel kondisi di dalam loop. Program akan macet dan tidak bisa berhenti sendiri. Kalau ini terjadi, tekan Ctrl + C di terminal untuk memaksanya berhenti. Selalu pastikan ada sesuatu di dalam loop yang pada akhirnya membuat kondisi menjadi False.

5.4 break, continue, dan pass

Ketiga kata kunci ini memberikan kendali yang lebih halus atas jalannya sebuah loop. Masing-masing punya peran yang berbeda dan sangat berguna dalam situasi tertentu. Memahami kapan memakai yang mana akan membuat kode kamu lebih bersih dan lebih efisien.

break — Hentikan Loop Sekarang

break langsung menghentikan loop dan melompat ke baris kode setelah loop tersebut, terlepas dari apakah kondisi loop masih terpenuhi atau belum. Gunakan ini ketika kamu sudah menemukan apa yang dicari dan tidak perlu melanjutkan iterasi — misalnya mencari data pertama yang memenuhi syarat tertentu dalam sebuah dataset:

python 
# Cari pelanggan pertama dengan skor kredit memenuhi syarat
data_pelanggan = [
    {"nama": "Andi",  "skor": 720},
    {"nama": "Budi",  "skor": 580},
    {"nama": "Citra", "skor": 810},
    {"nama": "Dian",  "skor": 750},
]

ditemukan = None
for p in data_pelanggan:
    if p["skor"] >= 800:
        ditemukan = p
        break    # berhenti segera, tidak perlu cek sisanya

if ditemukan:
    print(f"Ditemukan: {ditemukan['nama']} (skor: {ditemukan['skor']})")
    # → Ditemukan: Citra (skor: 810)
else:
    print("Tidak ada yang memenuhi syarat")

continue — Lewati Iterasi Ini, Lanjut ke Berikutnya

continue melewati sisa kode di iterasi saat ini dan langsung melompat ke iterasi berikutnya. Berbeda dari break yang menghentikan loop, continue hanya melewatkan satu iterasi. Sangat berguna untuk memfilter data yang tidak valid tanpa menghentikan pemrosesan keseluruhan:

python 
# Proses hanya data valid — lewati yang bermasalah
nilai_mentah = [85, None, 92, -5, 78, None, 110, 88]

nilai_valid = []
dilewati    = 0

for n in nilai_mentah:
    if n is None or not (0 <= n <= 100):
        dilewati += 1
        continue    # lewati, pergi ke elemen berikutnya
    nilai_valid.append(n)

print(f"Data valid   : {nilai_valid}")
# → Data valid   : [85, 92, 78, 88]
print(f"Data dilewati: {dilewati} entri bermasalah")
# → Data dilewati: 4 entri bermasalah
print(f"Rata-rata    : {sum(nilai_valid)/len(nilai_valid):.1f}")
# → Rata-rata    : 85.8

pass — Lakukan Tidak Ada

pass adalah kata kunci yang secara harfiah tidak melakukan apapun. Ia ada karena Python tidak mengizinkan blok kode yang kosong — setiap blok if, for, fungsi, atau kelas harus berisi setidaknya satu baris kode. pass berfungsi sebagai placeholder saat kamu sedang membangun kerangka *(skeleton)* kode dan belum siap mengisi implementasinya:

python 
# pass sebagai placeholder — kerangka kode yang belum diisi
def latih_model(data, config):
    pass    # TODO: implementasi pelatihan

def evaluasi_model(model, data_uji):
    pass    # TODO: implementasi evaluasi

def simpan_model(model, path):
    pass    # TODO: implementasi penyimpanan

# Kode di atas bisa dijalankan tanpa error
# pass memungkinkan kamu merancang struktur dulu, isi belakangan

# pass juga berguna di dalam kondisi saat satu kasus sengaja diabaikan
for kode in [200, 404, 500, 200]:
    if kode == 200:
        print(f"OK: {kode}")
    elif kode == 404:
        pass    # untuk sekarang abaikan 404
    else:
        print(f"Error: {kode}")
Kata Kunci Efek pada Loop Kapan Dipakai
break Hentikan loop sepenuhnya, lanjut setelah loop Sudah menemukan yang dicari, tidak perlu lanjut
continue Lewati iterasi ini, lanjut ke iterasi berikutnya Elemen saat ini tidak valid / tidak perlu diproses
pass Tidak melakukan apapun Placeholder untuk blok yang belum diimplementasi

5.5 Pemahaman Daftar (List Comprehension)

List comprehension adalah salah satu fitur Python yang paling elegan dan paling sering dipakai dalam pekerjaan data. Ia memungkinkan kamu membuat list baru dari koleksi lain dalam satu baris kode yang singkat dan mudah dibaca, tanpa harus menulis for loop yang panjang ditambah perintah append.

Sintaksis dasarnya adalah [ekspresi for item in koleksi]. Kamu bisa membacanya hampir seperti kalimat bahasa Inggris: "buat list yang berisi ekspresi untuk setiap item dalam koleksi." Opsional, tambahkan kondisi filter: [ekspresi for item in koleksi if kondisi] — artinya hanya item yang memenuhi kondisi yang diproses.

python 
# Perbandingan: for loop biasa vs list comprehension

# ── Cara lama: for loop + append ───────────────
kuadrat_lama = []
for n in range(1, 6):
    kuadrat_lama.append(n ** 2)

# ── Cara modern: list comprehension ────────────
kuadrat_baru = [n ** 2 for n in range(1, 6)]

print(kuadrat_lama)  # → [1, 4, 9, 16, 25]
print(kuadrat_baru)  # → [1, 4, 9, 16, 25]  ← hasil sama, kode lebih ringkas

Filter dengan Kondisi if

Tambahkan kondisi if di akhir untuk memfilter elemen. Ini menggantikan kombinasi for + if + append yang membutuhkan empat baris menjadi satu baris yang langsung bisa dibaca. Ini pola yang sangat umum dalam pembersihan data *(data cleaning)*:

python 
# Filter: hanya angka genap dari 1 sampai 10
genap = [n for n in range(1, 11) if n % 2 == 0]
print(genap)  # → [2, 4, 6, 8, 10]

# Ambil nilai yang lulus saja
semua_nilai  = [85, 45, 92, 38, 78, 55, 88]
nilai_lulus  = [n for n in semua_nilai if n >= 60]
print(nilai_lulus)  # → [85, 92, 78, 88]

# Konversi tipe data sekaligus — sangat sering di preprocessing
nilai_string = ["85", "92", "78", "95"]
nilai_int    = [int(n) for n in nilai_string]
print(nilai_int)   # → [85, 92, 78, 95]
print(f"Rata-rata: {sum(nilai_int) / len(nilai_int):.1f}")
# → Rata-rata: 87.5

# Transformasi teks — bersihkan dan standarisasi
ulasan_kotor  = ["  Produk Bagus! ", "SANGAT MEMBANTU", "  kurang memuaskan  "]
ulasan_bersih = [u.strip().lower() for u in ulasan_kotor]
print(ulasan_bersih)
# → ['produk bagus!', 'sangat membantu', 'kurang memuaskan']

Pola Nyata dalam Proyek Kecerdasan Buatan

List comprehension sangat bersinar dalam pekerjaan data dan kecerdasan buatan. Berikut empat pola yang akan kamu temui berulang kali dalam proyek nyata — simpan baik-baik karena kamu akan sering membutuhkannya:

python — pola nyata di ML 
# POLA 1: Normalisasi nilai ke rentang 0–1 (min-max scaling)
nilai_mentah   = [20, 50, 80, 35, 95]
min_val        = min(nilai_mentah)
max_val        = max(nilai_mentah)
ternormalisasi = [(n - min_val) / (max_val - min_val) for n in nilai_mentah]
print([f"{v:.2f}" for v in ternormalisasi])
# → ['0.00', '0.40', '0.80', '0.20', '1.00']

# POLA 2: Konversi probabilitas ke label biner
probabilitas = [0.2, 0.75, 0.6, 0.1, 0.9]
label        = [1 if p >= 0.5 else 0 for p in probabilitas]
print(label)  # → [0, 1, 1, 0, 1]

# POLA 3: Ekstrak satu kolom dari list of dictionaries
dataset    = [
    {"nama": "Andi",  "gaji": 8_000_000},
    {"nama": "Budi",  "gaji": 12_000_000},
    {"nama": "Citra", "gaji": 6_500_000},
]
semua_nama = [d["nama"] for d in dataset]
semua_gaji = [d["gaji"] for d in dataset]
print(semua_nama)
# → ['Andi', 'Budi', 'Citra']
print(f"Rata-rata gaji: Rp {sum(semua_gaji)/len(semua_gaji):,.0f}")
# → Rata-rata gaji: Rp 8,833,333

# POLA 4: Filter data yang memenuhi kriteria tertentu
gaji_tinggi = [d for d in dataset if d["gaji"] > 8_000_000]
print([d["nama"] for d in gaji_tinggi])
# → ['Budi']

Dictionary dan Set Comprehension

Python juga mendukung comprehension untuk dictionary dan set. Dictionary comprehension menggunakan sintaksis {kunci: nilai for item in koleksi}, sedangkan set comprehension cukup {ekspresi for item in koleksi}. Keduanya bekerja persis sama dengan list comprehension, hanya tipe hasilnya yang berbeda:

python 
# Dictionary comprehension
kata_kata    = ["python", "belajar", "data"]
peta_panjang = {kata: len(kata) for kata in kata_kata}
print(peta_panjang)
# → {'python': 6, 'belajar': 7, 'data': 4}

# Membalik kunci dan nilai dictionary
kode_kota = {"JKT": "Jakarta", "BDG": "Bandung", "SBY": "Surabaya"}
kota_kode  = {v: k for k, v in kode_kota.items()}
print(kota_kode)
# → {'Jakarta': 'JKT', 'Bandung': 'BDG', 'Surabaya': 'SBY'}

# Set comprehension — otomatis hilangkan duplikat
kalimat   = "belajar python dan belajar data science"
kata_unik = {kata for kata in kalimat.split()}
print(kata_unik)
# → {'belajar', 'python', 'dan', 'data', 'science'}  ← 'belajar' hanya sekali
✅
Kapan List Comprehension, Kapan for Loop Biasa?

Gunakan list comprehension untuk transformasi dan filter sederhana yang menghasilkan list baru — lebih cepat dan lebih ringkas. Pakai for loop biasa ketika logikanya kompleks (banyak kondisi bersarang atau banyak operasi per elemen), ketika kamu butuh efek samping selain membuat list, atau ketika comprehension-nya sudah terlalu panjang untuk dibaca dalam satu baris. Kode yang mudah dibaca selalu lebih penting dari kode yang sekadar singkat.
