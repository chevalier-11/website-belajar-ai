3.1 Variabel dan Tipe Data Dasar

Bayangkan variabel seperti sebuah kotak berlabel. Kamu bisa menaruh sesuatu ke dalam kotak itu, memberinya nama, lalu memanggil isi kotak itu kapanpun kamu mau lewat namanya. Dalam Python, membuat variabel semudah menuliskan nama, tanda sama dengan, lalu nilai yang ingin kamu simpan. Tidak perlu deklarasi tipe, tidak perlu kata kunci khusus — langsung saja.

python 
# Membuat variabel — sesederhana ini
nama       = "Sari"
usia       = 22
tinggi     = 165.5
sudah_lulus = True

# Memanggil variabel
print(nama)        # → Sari
print(usia)        # → 22
print(tinggi)      # → 165.5
print(sudah_lulus) # → True

Python adalah bahasa yang bertipe dinamis *(dynamically typed)*. Ini berarti kamu tidak perlu memberitahu Python bahwa nama adalah teks atau usia adalah angka — Python cukup pintar untuk menebaknya sendiri dari nilai yang kamu berikan. Ini membuat penulisan kode jadi lebih cepat, tapi juga menuntut kamu lebih hati-hati karena kesalahan tipe data baru terdeteksi saat kode dijalankan, bukan saat ditulis.

Kamu bisa memeriksa tipe sebuah variabel menggunakan fungsi bawaan type(). Ini sangat berguna saat men-debug kode yang bermasalah:

python 
print(type(nama))        # → <class 'str'>
print(type(usia))        # → <class 'int'>
print(type(tinggi))      # → <class 'float'>
print(type(sudah_lulus)) # → <class 'bool'>

Ada empat tipe data dasar yang baru saja kamu lihat: str untuk teks *(string)*, int untuk bilangan bulat *(integer)*, float untuk bilangan desimal, dan bool untuk nilai benar/salah *(boolean)*. Keempat ini adalah yang paling sering kamu temui sehari-hari, dan masing-masing akan dibahas lebih dalam di subbab-subbab berikutnya.

Aturan Penamaan Variabel

Nama variabel di Python punya beberapa aturan yang harus diikuti, plus konvensi yang disepakati industri. Aturan wajib: nama hanya boleh mengandung huruf, angka, dan garis bawah _; tidak boleh dimulai dengan angka; dan tidak boleh menggunakan kata kunci bawaan Python seperti if, for, print, dll.

Nama Variabel Valid? Keterangan
nama_pengguna ✅ Ya Standar — gunakan garis bawah untuk pemisah kata
NamaPengguna ✅ Ya Valid, tapi di Python dipakai khusus untuk nama kelas
jumlah2 ✅ Ya Angka boleh, tapi bukan di awal
2jumlah ❌ Tidak Tidak boleh dimulai dengan angka
nama-pengguna ❌ Tidak Tanda hubung dianggap operasi pengurangan
for ❌ Tidak Kata kunci bawaan Python tidak bisa dipakai

Konvensi standar Python — yang dikenal sebagai gaya snake_case — adalah menggunakan huruf kecil semua dengan garis bawah sebagai pemisah kata: nama_pengguna, total_harga, jumlah_data_latih. Gunakan nama yang deskriptif dan jelas maknanya, bukan singkatan yang membingungkan. Variabel bernama x atau tmp mungkin kamu mengerti sekarang, tapi tiga bulan kemudian saat kamu membaca ulang kodenya, kamu akan bingung sendiri.

Tes sederhana: bisakah seseorang yang belum pernah melihat kode kamu memahami apa yang disimpan dalam variabel itu hanya dari namanya? Kalau jawabannya tidak, ganti namanya. tinggi_badan_cm jauh lebih baik dari tb atau h.

Variabel Bisa Diubah Nilainya

Tidak seperti beberapa bahasa pemrograman lain, variabel di Python bisa diubah nilainya kapan saja — bahkan ke tipe data yang berbeda. Ini adalah konsekuensi dari sifat dinamis Python yang disebutkan tadi:

python 
skor = 85
print(skor, type(skor))  # → 85 <class 'int'>

skor = 92.5
print(skor, type(skor))  # → 92.5 <class 'float'>

skor = "sempurna"
print(skor, type(skor))  # → sempurna <class 'str'>

# Python tidak komplain — ini fitur, bukan bug
# Tapi dalam praktik, hindari mengubah tipe variabel
# karena membuat kode susah dipahami

3.2 String dan Operasi pada Teks

String adalah tipe data untuk menyimpan teks — apapun yang diapit oleh tanda kutip, baik kutip tunggal ' maupun kutip ganda ", adalah string. Di dunia kecerdasan buatan, string sangat penting karena kita sering bekerja dengan data teks: nama pengguna, komentar, transaksi, atau bahkan dokumen panjang yang diproses oleh model bahasa.

python 
# Berbagai cara membuat string
kalimat_satu = 'Ini string dengan kutip tunggal'
kalimat_dua  = "Ini string dengan kutip ganda"

# String multi-baris menggunakan tiga kutip
paragraf = """Ini adalah string
yang bisa memanjang
ke beberapa baris."""

# Keduanya tipe yang sama
print(type(kalimat_satu))  # → <class 'str'>
print(type(kalimat_dua))   # → <class 'str'>

Operasi Dasar pada String

Python memberikan banyak cara untuk memanipulasi string. Yang paling sering dipakai adalah penggabungan *(concatenation)*, pengulangan, dan pengukuran panjang:

python 
depan = "Budi"
belakang = "Santoso"

# Penggabungan string dengan operator +
nama_lengkap = depan + " " + belakang
print(nama_lengkap)  # → Budi Santoso

# Pengulangan string dengan operator *
garis = "-" * 30
print(garis)  # → ------------------------------

# Panjang string dengan fungsi len()
print(len(nama_lengkap))  # → 12
print(len("python"))      # → 6

f-String: Cara Terbaik Membuat Teks Dinamis

Salah satu fitur Python yang paling berguna dan paling sering kamu gunakan adalah f-string — cara menyisipkan nilai variabel langsung ke dalam teks tanpa perlu penggabungan yang panjang. Cukup tambahkan huruf f sebelum tanda kutip, lalu gunakan kurung kurawal {} untuk menyisipkan ekspresi:

python 
nama  = "Dina"
usia  = 24
kota  = "Bandung"
nilai = 87.6

# Cara lama — repot dengan penggabungan
pesan_lama = "Halo, " + nama + "! Usiamu " + str(usia) + " tahun."

# Cara modern — f-string, jauh lebih bersih
pesan_baru = f"Halo, {nama}! Usiamu {usia} tahun dan tinggal di {kota}."
print(pesan_baru)
# → Halo, Dina! Usiamu 24 tahun dan tinggal di Bandung.

# Bisa juga menyisipkan ekspresi langsung
print(f"Nilai kamu: {nilai:.1f}")  # → Nilai kamu: 87.6
print(f"Dua kali usia: {usia * 2}") # → Dua kali usia: 48
💡
Tentang Format Angka di f-String

Notasi {nilai:.1f} berarti "tampilkan variabel nilai sebagai desimal dengan 1 angka di belakang koma". Kamu bisa ubah angkanya: :.2f untuk 2 desimal, :.0f untuk tanpa desimal. Ini sangat berguna saat menampilkan hasil perhitungan atau metrik model.

Metode Bawaan String yang Penting

String di Python punya banyak metode bawaan yang bisa langsung dipanggil. Metode adalah fungsi yang "melekat" pada sebuah objek — cara memanggilnya adalah variabel.metode(). Berikut yang paling sering digunakan:

python 
teks = "  Belajar Python itu Menyenangkan!  "

# Mengubah huruf besar/kecil
print(teks.upper())    # → "  BELAJAR PYTHON ITU MENYENANGKAN!  "
print(teks.lower())    # → "  belajar python itu menyenangkan!  "
print(teks.title())    # → "  Belajar Python Itu Menyenangkan!  "

# Menghapus spasi di awal dan akhir
print(teks.strip())    # → "Belajar Python itu Menyenangkan!"

# Memeriksa isi string
print(teks.startswith("  Belajar"))  # → True
print("Python" in teks)             # → True

# Mengganti bagian teks
baru = teks.replace("Python", "Coding")
print(baru)  # → "  Belajar Coding itu Menyenangkan!  "

# Memecah string menjadi daftar kata
kalimat = "satu dua tiga empat"
kata_kata = kalimat.split()
print(kata_kata)  # → ['satu', 'dua', 'tiga', 'empat']

# Memecah berdasarkan karakter tertentu
tanggal = "2024-12-25"
bagian  = tanggal.split("-")
print(bagian)  # → ['2024', '12', '25']

Pengindeksan dan Pengirisan String

Kamu bisa mengakses karakter individual dalam sebuah string menggunakan indeks — posisi karakter dalam teks, dimulai dari 0 untuk karakter pertama. Python juga mendukung indeks negatif: -1 untuk karakter terakhir, -2 untuk yang kedua dari belakang, dan seterusnya. Untuk mengambil sebagian teks, gunakan teknik pengirisan *(slicing)* dengan sintaksis [mulai:akhir]:

python 
bahasa = "Python"
#          P  y  t  h  o  n
# indeks:  0  1  2  3  4  5
# negatif:-6 -5 -4 -3 -2 -1

# Akses karakter tunggal
print(bahasa[0])   # → P
print(bahasa[3])   # → h
print(bahasa[-1])  # → n (karakter terakhir)

# Pengirisan [mulai:akhir] — akhir tidak ikut
print(bahasa[0:3])  # → Pyt
print(bahasa[2:])   # → thon (dari indeks 2 sampai akhir)
print(bahasa[:4])   # → Pyth (dari awal sampai indeks 3)
print(bahasa[::-1]) # → nohtyP (dibalik)
⚠️
String Tidak Bisa Diubah Sebagian

String di Python bersifat tidak bisa diubah *(immutable)* — kamu tidak bisa mengubah karakter individual dengan bahasa[0] = "J". Kalau mau mengubah isi string, kamu harus membuat string baru dari yang lama, misalnya dengan .replace() atau f-string. Ini berbeda dari tipe data seperti list yang akan kita pelajari di bab berikutnya.

3.3 Angka: Integer, Float, dan Operasi Matematika

Python punya dua tipe data angka utama yang perlu kamu bedakan: integer (int) untuk bilangan bulat tanpa koma seperti 1, 42, atau -100, dan float (float) untuk bilangan desimal seperti 3.14, 0.001, atau -7.5. Pembedaan ini penting karena perilaku matematikanya bisa berbeda dalam beberapa situasi.

python 
# Integer — bilangan bulat
populasi  = 270000000
suhu_min  = -10
nol       = 0

# Float — bilangan desimal
pi       = 3.14159
gravitasi = 9.8
persen    = 0.85

# Python mendukung angka besar tanpa batas (dengan tanda _ untuk keterbacaan)
besar = 1_000_000_000  # sama dengan 1000000000, lebih mudah dibaca
print(besar)  # → 1000000000

Operator Matematika di Python

Python mendukung semua operator aritmatika standar, plus beberapa yang mungkin belum kamu kenal. Semua ini akan sangat sering kamu gunakan saat bekerja dengan data dan model kecerdasan buatan:

Operator Nama Contoh Hasil
+ Penjumlahan 10 + 3 13
- Pengurangan 10 - 3 7
* Perkalian 10 * 3 30
/ Pembagian (selalu float) 10 / 3 3.3333...
// Pembagian bulat 10 // 3 3
% Sisa bagi (modulo) 10 % 3 1
** Pangkat 10 ** 3 1000

Dari tabel di atas, ada dua operator yang mungkin asing bagimu. Operator // adalah pembagian bulat — hasilnya selalu dibulatkan ke bawah, membuang bagian desimal. Operator % adalah modulo — menghasilkan sisa dari pembagian. Keduanya sangat sering muncul dalam algoritma pemrograman: modulo misalnya berguna untuk mengecek apakah sebuah angka genap atau ganjil (angka % 2 == 0 berarti genap).

python 
# Contoh penggunaan operator yang lebih kompleks
nilai_ujian  = [85, 90, 78, 92, 88]
jumlah_nilai = 85 + 90 + 78 + 92 + 88
rata_rata    = jumlah_nilai / 5
print(f"Rata-rata: {rata_rata:.1f}")  # → Rata-rata: 86.6

# Cek apakah angka genap atau ganjil
angka = 17
if angka % 2 == 0:
    print(f"{angka} adalah genap")
else:
    print(f"{angka} adalah ganjil")  # → 17 adalah ganjil

# Pangkat — sering dipakai dalam matematika ML
kuadrat = 5 ** 2   # → 25
akar_dua = 16 ** 0.5  # → 4.0 (akar kuadrat via pangkat 0.5)
print(kuadrat, akar_dua)

Operator Penugasan Campuran

Sangat sering dalam pemrograman kamu perlu mengubah nilai variabel berdasarkan nilainya sendiri — misalnya menambahkan 1 ke penghitung, atau mengalikan sesuatu dengan konstanta. Python punya cara singkat untuk ini yang menghemat banyak pengetikan:

python 
penghitung = 10

penghitung += 5   # sama dengan: penghitung = penghitung + 5
print(penghitung)   # → 15

penghitung -= 3   # sama dengan: penghitung = penghitung - 3
print(penghitung)   # → 12

penghitung *= 2   # sama dengan: penghitung = penghitung * 2
print(penghitung)   # → 24

penghitung //= 4  # sama dengan: penghitung = penghitung // 4
print(penghitung)   # → 6
📌
Float dan Presisi Angka

Float di komputer disimpan dalam format biner, yang kadang menyebabkan hasil yang sedikit mengejutkan: 0.1 + 0.2 di Python menghasilkan 0.30000000000000004, bukan persis 0.3. Ini bukan bug Python — ini sifat mendasar cara komputer menyimpan angka desimal. Untuk perhitungan keuangan atau saat presisi sangat penting, gunakan modul decimal bawaan Python.

3.4 Nilai Boolean dan Logika Dasar

Boolean adalah tipe data paling sederhana dalam pemrograman: hanya ada dua kemungkinan nilai, True (benar) atau False (salah). Meski tampak simpel, boolean adalah tulang punggung dari semua logika pengambilan keputusan dalam program — setiap kondisi if, setiap perulangan bersyarat, dan setiap filter data pada akhirnya bergantung pada nilai boolean.

python 
# Boolean langsung
lulus       = True
sudah_bayar = False

print(type(lulus))  # → <class 'bool'>

# Boolean dari perbandingan — ini yang paling sering terjadi
nilai = 75
print(nilai >= 60)   # → True
print(nilai == 100)  # → False
print(nilai != 75)   # → False (75 memang sama dengan 75)

Operator Perbandingan

Operator perbandingan menghasilkan nilai boolean — selalu True atau False. Ini yang paling sering kamu gunakan untuk membuat keputusan dalam kode:

python 
a = 10
b = 20

print(a == b)   # → False  (sama dengan)
print(a != b)   # → True   (tidak sama dengan)
print(a < b)    # → True   (kurang dari)
print(a > b)    # → False  (lebih dari)
print(a <= b)   # → True   (kurang dari atau sama dengan)
print(a >= b)   # → False  (lebih dari atau sama dengan)

# Perbandingan juga bisa berantai — ini fitur keren Python
nilai = 75
print(60 <= nilai < 80)  # → True (apakah nilai antara 60 dan 79?)
⚠️
Jangan Bingung = dan ==

Ini adalah kesalahan paling klasik bagi pemula. Satu tanda sama dengan = adalah untuk memberi nilai ke variabel (penugasan). Dua tanda sama dengan == adalah untuk membandingkan apakah dua nilai sama (perbandingan). Menulis if nilai = 75: akan langsung menghasilkan error. Yang benar adalah if nilai == 75:.

Operator Logika: and, or, not

Seringkali kamu butuh menggabungkan lebih dari satu kondisi. Python menyediakan tiga operator logika untuk ini: and (dan — kedua kondisi harus benar), or (atau — cukup salah satu yang benar), dan not (bukan — membalik nilai boolean):

python 
nilai       = 75
sudah_hadir = True
sudah_bayar = False

# and — kedua kondisi harus True
bisa_lulus = (nilai >= 60) and sudah_hadir
print(f"Bisa lulus: {bisa_lulus}")   # → True

# or — cukup satu kondisi yang True
perlu_tindak = (nilai < 60) or (not sudah_bayar)
print(f"Perlu tindak: {perlu_tindak}")  # → True (karena belum bayar)

# not — membalik nilai boolean
print(not sudah_bayar)  # → True
print(not sudah_hadir)  # → False

# Menggabungkan beberapa kondisi
lolos_semua = (nilai >= 60) and sudah_hadir and sudah_bayar
print(f"Lolos semua: {lolos_semua}")   # → False (belum bayar)

Nilai yang Dianggap False

Di Python, selain nilai False itu sendiri, ada beberapa nilai lain yang dianggap "salah" ketika digunakan dalam konteks boolean. Memahami ini penting karena sering muncul dalam kondisi if:

  • 0 dan 0.0 — angka nol dianggap False
  • "" — string kosong dianggap False
  • [], {}, () — koleksi kosong dianggap False
  • None — nilai khusus Python yang berarti "tidak ada nilai" dianggap False
python 
# Semua ini dianggap False dalam konteks boolean
print(bool(0))        # → False
print(bool(""))       # → False
print(bool([]))       # → False
print(bool(None))     # → False

# Semua ini dianggap True
print(bool(1))        # → True
print(bool("halo"))  # → True
print(bool([1,2,3])) # → True

# Praktis dipakai untuk mengecek apakah daftar tidak kosong
data = [10, 20, 30]
if data:
    print("Data ada isinya!")  # → ini yang akan tercetak

3.5 Konversi Antar Tipe Data

Dalam praktik pemrograman sehari-hari, kamu akan sangat sering berurusan dengan data yang punya tipe yang tidak sesuai dengan apa yang kamu butuhkan. Misalnya, kamu membaca angka dari file teks — hasilnya adalah string, bukan integer. Atau kamu ingin menampilkan angka float dengan hanya beberapa desimal. Di sinilah konversi tipe data *(type conversion)* menjadi sangat berguna.

Python menyediakan fungsi bawaan untuk mengkonversi antara tipe data: int(), float(), str(), dan bool(). Proses ini disebut konversi eksplisit *(explicit conversion)* atau kadang disebut pengecoran tipe *(type casting)*:

python 
# str → int (sangat sering terjadi saat baca input pengguna)
umur_teks = "24"
umur_angka = int(umur_teks)
print(umur_angka + 1)  # → 25 (bisa dihitung sekarang)

# str → float
harga_teks  = "19.99"
harga_angka = float(harga_teks)
print(harga_angka * 2)  # → 39.98

# int → float dan sebaliknya
bilangan_bulat  = 7
bilangan_desimal = float(bilangan_bulat)
print(bilangan_desimal)  # → 7.0

kembali_bulat = int(3.99)
print(kembali_bulat)     # → 3 (dibuang, BUKAN dibulatkan)

# int/float → str (saat perlu menggabungkan angka dengan teks)
skor      = 95
pesan     = "Skor kamu: " + str(skor)
print(pesan)  # → Skor kamu: 95

# Cara lebih baik: pakai f-string (tidak perlu konversi manual)
pesan_fstring = f"Skor kamu: {skor}"
print(pesan_fstring)  # → Skor kamu: 95
⚠️
Konversi Bisa Gagal — Selalu Waspada

Tidak semua konversi bisa berhasil. Mencoba int("halo") atau float("abc") akan menghasilkan error ValueError. Dalam program nyata, kamu perlu menangani kemungkinan ini — biasanya dengan blok try-except yang akan dibahas di Bab 7. Selalu validasi data sebelum mengkonversinya, terutama data yang datang dari pengguna atau file eksternal.

Konversi Implisit

Selain konversi eksplisit yang kamu lakukan sendiri, Python kadang melakukan konversi secara otomatis di balik layar. Ini disebut konversi implisit *(implicit conversion)*. Contoh paling umum adalah saat kamu mengoperasikan integer dan float bersama-sama — Python otomatis mengubah integer menjadi float untuk menjaga presisi hasil:

python 
# Python otomatis mempromosikan int ke float saat perlu
hasil = 5 + 2.0
print(hasil)         # → 7.0 (bukan 7)
print(type(hasil))   # → <class 'float'>

# Pembagian biasa selalu menghasilkan float
print(10 / 2)  # → 5.0  (float, bukan 5)
print(10 // 2) # → 5   (int, karena pembagian bulat)

# Tapi Python TIDAK otomatis mengkonversi string ke angka
# Baris berikut akan error:
# print("5" + 3)  # → TypeError!
# Harus eksplisit:
print(int("5") + 3)   # → 8
print("5" + str(3))   # → "53" (penggabungan string!)

Fungsi input() dan Konversi Tipe

Satu situasi sangat umum yang hampir selalu membutuhkan konversi tipe adalah pengambilan masukan dari pengguna lewat fungsi input(). Apapun yang pengguna ketik, input() selalu mengembalikannya sebagai string — bahkan jika yang diketik adalah angka. Kamu harus selalu mengkonversinya sesuai kebutuhan:

python 
# input() selalu mengembalikan string
nama_input  = input("Siapa namamu? ")
# → pengguna mengetik: Rani
print(f"Halo, {nama_input}!")  # → Halo, Rani!

# Untuk angka, harus dikonversi dulu
usia_teks  = input("Berapa usiamu? ")
# → pengguna mengetik: 22
usia_angka = int(usia_teks)  # konversi ke int
tahun_lahir = 2024 - usia_angka
print(f"Kamu lahir sekitar tahun {tahun_lahir}.")
# → Kamu lahir sekitar tahun 2002.

# Cara ringkas: konversi langsung saat input
tinggi = float(input("Tinggi badanmu (cm)? "))
print(f"Tinggi: {tinggi:.1f} cm")
✅
Rangkuman Konversi Tipe yang Sering Dipakai

Hafal empat ini dan kamu sudah siap: int(x) untuk mengubah ke bilangan bulat — bagian desimal dibuang. float(x) untuk mengubah ke bilangan desimal. str(x) untuk mengubah ke teks — hampir semua tipe bisa diubah ke string. bool(x) untuk mengubah ke True atau False — ingat, 0, string kosong, dan None dianggap False.
